1. Устанавливаем IIS через web platform installer или в установке компонентов.
2. Устанавливаем IIS asp.net.
3. Устанавливаем SQL server 2012 (express) для работы с LocalDB.
4. В диспетчере IIS добавляем новый сайт, указываем имя, физ. каталог нашего сайта, привязываем IP, домен, среда пула .net 4.
5. Выставляем права на app_data (для бд) MSSQLEXPRESS на запись, возможно потребуется удалить .log файл в данной папке.
6. Возможно требуется изменить пароль от пользователя sa в webconfig и data source sql сервера.
7. Заходим по ip или домену, регистрируемся.
